Каскадные таблицы стилей: CSS

Каскадные таблицы стилей описывают внешний вид документа, написанного с помощью языка разметки (HTML).

CSS-свойства

С помощью CSS-свойств задают внешний вид страницы: размер шрифта, его начертание, размеры блоков, способы их отображения, отступы, цвет текста, создают анимацию и многое другое.
список свойств CSS
CSS-свойства можно задавать несколькими способами:

  • в атрибуте style HTML-тега
  • с помощью правил внутри тега style
  • с помощью правил в подключенном файле с разрешением *.css
CSS-свойства задаются таким образом: свойство: значение;
Правила хорошего тона предписывают нам НЕ задавать CSS-свойства с помощью атрибута style без необходимости.

CSS-правила и селекторы

Правила CSS объединяют в себе набор свойств и выглядят так:

          селектор, селектор... {
            свойство: значение;
            свойство: значение;
            ...
          }

Селекторы бывают разных видов и определяют, для каких элементов определить описанные свойства.
список селекторов CSS
Основные виды селекторов:

  • Универсальный селектор - устанавливает стиль для всех элементов страницы. 
                  * {
                ...
              }
  • Селектор тегов - устанавливает стиль для всех тегов с заданным именем.
    В примере для всех тегов h2 задан цвет текста. 
                  h2 {
                color: #5E8DB2;
              }
  • Селектор классов - устанавливает стиль для всех тегов с заданным классом.
    Класс тега задается с помощью атрибута class. 
                  .code {
                font-family: 'Courier New', Courier, 'Lucida Sans Typewriter', 'Lucida Typewriter', monospace;
                font-weight: bold;
                color: #FF5722;
                font-size: 1.2em;
              }


              span class="code"class/span
  • Селектор идентификаторов - устанавливает стиль для всех тегов с заданным идентификатором.
    Идентификатор тега задается с помощью атрибута id. 
                  #tag-types {
                text-decoration: underline;
              }
  • Селектор атрибутов - устанавливает стиль для всех тегов с определенным значением указанного атрибута.
    В примере для всех тегов a с атрибутом target="_blank" устанавливается цвет текста. 
                  a[target="_blank"] {
                color: #109684;
              }
  • Селектор потомков - устанавливает стиль для всех элементов, являющимися потомками заданного элемента независимо от вложенности.
    В примере: для всех тегов span, являющихся потомками тега ul с классом list-none установлен жирный шрифт. 
                  ul.list-none span {
                font-weight: bold;
              }
  • Селектор дочерних элементов - устанавливает стиль для всех элементов, являющимися прямыми потомками заданного элемента.
    В примере: для всех тегов li, являющихся прямыми потомками тега ul с классом list-none установлен цвет текста для первой буквы. 
                  ul.list-none>li::first-letter {
                color: #5E8DB2;
              }
  • Селектор псевдоклассов - устанавливает стиль для различных состояний элементов, например, для ссылки при наведении на нее курсора. 
                  a:hover {
                color: #5ADAC9;
              }
  • Селектор псевдоэлементов - устанавливает стиль элементов, отсутствующих в дереве документа. Например, можно определить стиль первой буквы содержимого элемента или дописать контент до или после содержимого элемента. 
                  ul.list-none>li::first-letter {
                color: #5E8DB2;
              }
              .tag::before {
                content: "<";
              }
              .tag::after {
                content: ">";
              }
